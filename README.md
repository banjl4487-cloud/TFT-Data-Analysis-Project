🏆  tft VI 챔피언 데이터 심층 분석: 방어 아이템 기반 승률 최적화 전략 제안
🎯 1. 프로젝트 개요: 데이터로 증명하는 VI의 최적화된 힘
이 프로젝트는 인기 게임 tft 즉 "Teamfight Tactics" 내 'VI' 챔피언에 집중하여, 방어 아이템(Defensive Items) 선택이 챔피언의 승률에 미치는 영향을 데이터 과학적 방법론으로 심층 분석합니다. VI 플레이어의 아이템 빌드 최적화 및 메타 포지셔닝에 대한 실질적인 인사이트를 제공하여, 데이터 기반 의사결정의 중요성을 증명합니다. 단순한 경험을 넘어, 데이터를 통해 문제 정의-분석-해결 프로세스를 구현한 프로젝트입니다.

✨ 2. 주요 분석 목표 및 결과: VI를 무적의 전략 병기로
데이터 구축 및 정제: Kaggle LoL 데이터셋에서 VI 챔피언 플레이 데이터 추출, 전처리 및 구조화하여 고품질 분석 데이터셋을 완성했습니다.
아이템 분류 및 메타 분석: VI가 사용하는 아이템을 '방어'와 '비방어'로 명확히 분류하고, 시즌별 아이템 메타 트렌드를 파악했습니다.

방어 아이템과 승률 상관관계 분석:
특정 방어 아이템 착용이 VI의 승률에 미치는 영향을 통계적으로 검증했습니다.
핵심 결과: (예시: "'얼어붙은 건틀릿'을 포함한 빌드는 타 방어 아이템 빌드 대비 VI의 승률을 평균 3.5%p 상승시켰으며, 이는 유의미한 생존력 증가로 이어짐을 확인했습니다.")
VI 최적 빌드 전략 제안: 분석 결과를 토대로 게임 상황별 VI 챔피언의 최적 방어 아이템 빌드 가이드라인을 제시하여, 실제 플레이에 적용 가능한 전략적 통찰을 제공합니다.
