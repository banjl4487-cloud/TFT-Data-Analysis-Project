🏆 프로젝트명: TFT VI 챔피언 데이터 심층 분석: 방어 아이템 기반 빌드 전략 제안
1. 프로젝트 개요: 데이터로 증명하는 VI의 최적화된 힘
본 프로젝트는 인기 게임 "Teamfight Tactics (TFT)" 내 'VI' 챔피언에 집중하여, 방어 아이템 선택이 챔피언의 빌드에 미치는 영향을 데이터 과학적 방법론으로 심층 분석합니다. VI 플레이어의 아이템 빌드 최적화 및 메타 포지셔닝에 대한 실질적인 인사이트를 제공하여, 데이터 기반 의사결정의 중요성을 증명합니다. 단순한 경험을 넘어, 저의 데이터를 통한 문제 정의-분석-해결 프로세스 구현 역량을 보여주는 전략적 분석 프로젝트입니다.

2. 주요 성과 및 기술적 역량 (Highlights)
고품질 데이터 구축 및 전처리:
Kaggle의 LoL 데이터셋에서 VI 챔피언 플레이 데이터를 추출하고, 철저한 전처리 및 구조화를 통해 고품질의 분석 데이터셋을 완성했습니다. 이는 대용량 데이터 수집 및 전처리 역량을 증명합니다.
아이템 분류 및 메타 동향 분석:
VI가 사용하는 아이템을 '방어'와 '비방어'로 명확히 분류하고, 시즌별 아이템 메타 트렌드를 파악하여 챔피언 빌드의 핵심 요소들을 식별했습니다.
방어 아이템과 챔피언 빌드 상관관계 분석 및 전략 제안:
특정 방어 아이템 착용이 VI 챔피언 빌드 구성과 역할군에 미치는 영향을 데이터로 증명했습니다. 분석 결과를 토대로 게임 상황별 VI 챔피언의 최적 아이템 빌드 가이드라인을 제시하여, 실제 플레이에 적용 가능한 전략적 통찰을 제공합니다.

핵심 통찰:
"'얼어붙은 건틀릿'을 포함한 빌드가 예상과는 달리 VI의 다양한 역할 (보조 탱커, 브루저) 빌드에서 높은 채택률을 보임"을 데이터로 입증, 기존 플레이 양상에 대한 새로운 관점을 제시했습니다.
이를 뒷받침하기 위해 VI 다음으로 많이 쓰였던 '블리츠크랭크' 챔피언의 아이템 분석까지 Python 시각화 라이브러리로 분석하고 첨부하여 분석의 객관성과 깊이를 강화했습니다. 이 시각화 자료를 통해 방어 아이템 선택에 따른 승률 변화와 빌드 트렌드를 직관적으로 확인할 수 있습니다.

주요 기술 스택: Python (Pandas, Numpy, Matplotlib, Seaborn), Git / GitHub

3. 향후 계획: 지속적인 성장과 데이터 기반 전략 강화 로드맵
본 프로젝트는 현재에 만족하지 않고, 게임 데이터 분석 역량 강화를 위한 저의 비전을 보여줍니다.
다양한 챔피언 및 아이템 메타 확장 분석: 특정 챔피언을 넘어 TFT의 전반적인 게임 메타 동향을 분석하여 광범위한 인사이트를 제공할 계획입니다.
승률 예측 모델 개발: 챔피언의 아이템 빌드에 따른 승률 예측 모델을 개발하여 플레이어의 전략적 의사결정을 더욱 정교하게 지원할 예정입니다.
실시간 데이터 연동 및 자동화: API 등을 활용하여 실시간 게임 데이터를 수집하고 분석 과정을 자동화하는 시스템을 구축하여, 분석의 시의성을 확보하는 것을 목표로 합니다.

4. 프로젝트 실행 가이드 (Execution Guide)

git clone https://github.com/banjl4487-cloud/TFT-Data-Analysis-Project
3. 분석 및 시각화 스크립트 실행:
